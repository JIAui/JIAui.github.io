PIP

pip就是python标准库(The Python Standard Library)中的一个包，用来管理Python标准库中的其他包。

usage

命令行直接键入pip -h即可查看用法：

使用pip安装第三方库

命令行执行pip install <package name>或pip install -r

卸载安装的第三方库

pip uninstall <package name>或pip uninstall -r requirements.txt

列出已安装的包

pip list

查询可升级的包

pip list -o

python3.8.0下载安装好后，无法使用pip的问题

  • cmd中到达安装目录下”python/Scripts”其中就有”pip.py”等文件.

  • 输入指令”pip”查看有关pip的详细信息(即运行了pip)

  • 最后将”pip.exe”所在目录添加到系统环境变量的Path变量中,例如:”E:\Program Files (x86)\Python_3.8.0\Scripts”。

    (可以将系统环境变量中的path项添加python/Scripts的路径)
